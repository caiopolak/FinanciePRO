Instalação e Configuração

Pré-requisitos

  • Python 3.8 ou superior
  • Pip (gerenciador de pacotes Python)
  • Conta no Supabase
  • Conta no Stripe (opcional para pagamentos)

Passo a Passo para Instalação Local

# Clone o repositório
git clone https://github.com/seu-usuario/FinanciePRO.git

# Entre na pasta do projeto
cd FinanciePRO

# Crie um ambiente virtual (recomendado)
python -m venv venv

# Ative o ambiente virtual
# Linux/Mac:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# Instale as dependências
pip install -r requirements.txt

# Configure as variáveis de ambiente
# Crie um arquivo .env na raiz do projeto com:
SUPABASE_URL="sua-url-supabase"
SUPABASE_KEY="sua-chave-supabase"
STRIPE_API_KEY="sua-chave-stripe"  # opcional

# Execute a aplicação
python run.py

# Acesse no navegador: http://localhost:5000

Deploy em Produção

Para colocar seu aplicativo online:

  1. Escolha um provedor de hospedagem (AWS, Google Cloud, Heroku, etc.)
  2. Configure um servidor web (Nginx ou Apache)
  3. Use o Gunicorn como servidor WSGI para aplicações Python
  4. Configure um domínio personalizado com HTTPS
  5. Monitore sua aplicação com ferramentas como Sentry ou New Relic
Importante: Nunca exponha suas chaves de API no código fonte. Utilize variáveis de ambiente ou serviços de gerenciamento de segredos.

Configuração do Supabase

Configuração Inicial

  1. Crie uma conta no Supabase
  2. Crie um novo projeto com as configurações:
    • Nome: FinanciePRO-Production
    • Região: South America (São Paulo)
    • Senha: Use uma senha forte e segura
  3. Acesse as configurações do projeto e obtenha:
    • URL: Adicione ao .env como SUPABASE_URL
    • Chave API: Adicione ao .env como SUPABASE_KEY

Estrutura do Banco de Dados

O FinanciePRO utiliza as seguintes tabelas:

-- Tabela de Usuários
CREATE TABLE users (
    id UUID PRIMARY KEY,
    email TEXT NOT NULL UNIQUE,
    password_hash TEXT NOT NULL,
    name TEXT NOT NULL,
    plan TEXT DEFAULT 'free',
    created_at TIMESTAMPTZ DEFAULT NOW(),
    currency TEXT DEFAULT 'R$',
    risk_profile TEXT DEFAULT 'moderate',
    notification_pref BOOLEAN DEFAULT TRUE
);

-- Tabela de Transações
CREATE TABLE transactions (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    type TEXT NOT NULL CHECK (type IN ('income', 'expense')),
    amount DECIMAL NOT NULL,
    category TEXT NOT NULL,
    description TEXT,
    date DATE NOT NULL,
    recurring BOOLEAN DEFAULT FALSE
);

-- Tabela de Metas
CREATE TABLE goals (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    name TEXT NOT NULL,
    target_amount DECIMAL NOT NULL,
    current_amount DECIMAL DEFAULT 0,
    target_date DATE NOT NULL,
    priority TEXT DEFAULT 'medium' CHECK (priority IN ('low', 'medium', 'high'))
);

-- Tabela de Investimentos
CREATE TABLE investments (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    name TEXT NOT NULL,
    type TEXT NOT NULL CHECK (type IN ('stocks', 'bonds', 'crypto', 'real_estate', 'funds', 'others')),
    amount DECIMAL NOT NULL,
    expected_return DECIMAL,
    date DATE NOT NULL,
    notes TEXT
);

-- Tabela de Orçamentos
CREATE TABLE budgets (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    category TEXT NOT NULL,
    amount DECIMAL NOT NULL,
    month INTEGER NOT NULL,
    year INTEGER NOT NULL
);

-- Tabela de Notificações
CREATE TABLE notifications (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    message TEXT NOT NULL,
    date TIMESTAMPTZ DEFAULT NOW(),
    read BOOLEAN DEFAULT FALSE
);

-- Tabela de Assinaturas
CREATE TABLE subscriptions (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    plan TEXT NOT NULL,
    status TEXT NOT NULL CHECK (status IN ('active', 'canceled', 'expired', 'pending')),
    start_date TIMESTAMPTZ NOT NULL,
    end_date TIMESTAMPTZ,
    payment_method TEXT NOT NULL,
    stripe_subscription_id TEXT
);

Função RPC Necessária

Crie esta função no Supabase para atualizar metas:

CREATE OR REPLACE FUNCTION increment_goal(goal_id UUID, inc_amount FLOAT)
RETURNS VOID AS $$
UPDATE goals
SET current_amount = current_amount + inc_amount
WHERE id = goal_id;
$$ LANGUAGE SQL;

Como Usar o FinanciePRO

Dashboard Principal

O dashboard oferece uma visão geral das suas finanças:

  • Saldo Atual: Seu saldo financeiro total
  • Receitas: Total de entradas no período
  • Despesas: Total de saídas no período
  • Gráficos: Visualização de receitas vs despesas e distribuição de gastos
  • Alertas: Notificações importantes sobre suas finanças

Gerenciando Transações

Para adicionar uma transação:

  1. Navegue até "Transações" no menu
  2. Clique em "Nova Transação"
  3. Preencha os campos:
    • Tipo (Receita/Despesa)
    • Valor
    • Data
    • Categoria
    • Descrição (opcional)
  4. Salve a transação

Criando e Acompanhando Metas

Para definir uma meta financeira:

  1. Acesse a seção "Metas"
  2. Clique em "Nova Meta"
  3. Defina:
    • Nome da meta
    • Valor alvo
    • Data limite
    • Prioridade
  4. Acompanhe o progresso na barra de progresso
  5. Atualize o valor atual conforme for economizando

Configurando Orçamentos

Para criar um orçamento mensal:

  1. Vá para "Orçamentos"
  2. Selecione o mês e ano
  3. Clique em "Novo Orçamento"
  4. Escolha uma categoria e defina um valor limite
  5. O sistema alertará quando você estiver próximo ou exceder o limite

Gerenciando Assinaturas

Para atualizar seu plano:

  1. Acesse "Assinatura" no menu
  2. Escolha entre os planos Gratuito, PRO ou Business
  3. Preencha os dados de pagamento
  4. Confirme a assinatura
  5. Aproveite os recursos adicionais do seu novo plano

Dicas Financeiras e Melhores Práticas

Regra 50/30/20

Distribua sua renda da seguinte forma:

  • 50% para necessidades (aluguel, alimentação, transporte)
  • 30% para desejos (lazer, compras não essenciais)
  • 20% para investimentos e poupança

Fundo de Emergência

Construa um fundo equivalente a:

  • 3-6 meses de suas despesas essenciais
  • Mantenha em investimento de alta liquidez
  • Atualize o valor anualmente

Investimentos Inteligentes

Diversifique seus investimentos:

  • Renda Fixa: Tesouro Direto, CDBs
  • Renda Variável: Ações, ETFs
  • Fundos Imobiliários
  • Criptomoedas (com moderação)

Desafios Financeiros

Desafio do Café

Reduza em 50% os gastos com cafés e lanches fora de casa durante 1 mês.

Desafio de 7 Dias

Fique 7 dias sem fazer nenhuma compra não essencial.

Desafio de R$1000

Encontre R$1000 em economias em 30 dias (cortando supérfluos).
